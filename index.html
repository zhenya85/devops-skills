● Фаза I — Базовый уровень
01
БАЗОВЫЙ Первый шаг — фундамент для всего курса
💡Рекомендуется начать именно с SQL и Python, так как курс «Старт в DevOps» предполагает знакомство с базами данных, а знание Python потребуется уже в первых практических заданиях.
🎯 Рекомендации по прохождению этапа
  • Оба курса проходите параллельно — SQL и Python не зависят друг от друга и хорошо чередуются.
  • По SQL сразу практикуйтесь в pgAdmin или DBeaver: пишите реальные запросы SELECT, JOIN, GROUP BY к тестовой БД.
  • По Python используйте REPL (интерактивную консоль) для каждой новой темы — не просто читайте код, набирайте его сами.
  • Установите PostgreSQL локально ещё до начала курса по SQL — так практика пойдёт быстрее.
  • Не задерживайтесь на «продвинутом SQL» слишком долго — базового уровня (SELECT, JOIN, подзапросы) достаточно для продолжения курса.

02
БАЗОВЫЙСистемное администрирование Linux — главный фундамент DevOps
🎯 Рекомендации по прохождению этапа
  • Разверните виртуальную машину Ubuntu (VirtualBox или VMware) — практикуйтесь только в живой среде, не в симуляторе.
  • Ведите личный «шпаргалочный файл» команд Bash: после каждого урока добавляйте 3–5 новых команд с примерами.
  • Раздел по сетям — один из сложнейших; не торопитесь, разберите модель OSI и TCP/IP на реальных примерах с tcpdump/netstat.
  • Vim достаточно знать на уровне базовых команд (вставка, сохранение, поиск) — не тратьте на него больше одного дня.
  • Завершите курс мини-проектом: напишите Bash-скрипт, который автоматически создаёт нового пользователя, задаёт права и отправляет уведомление в лог.

БАЗОВЫЙВведение в DevOps-практики и Docker
